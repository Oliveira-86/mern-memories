# Your Memories

#### The App is called "Memories" and it is a simple social media MERN application that allows users to post interesting events that happened in their lives.

### Deployment

- Heroku (Backend)
- Netlify (Frontend)

- [x] CRUD of post
- [x] Login with Email JWT and Google authetication
